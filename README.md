# rails_view_counter
> rails plugin new rails_view_counter --mountable --full

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "rails_view_counter"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_view_counter
```
